# Quickstart: Phase-1 Todo App

## Prerequisites
- Python 3.13 or higher
- Console/terminal access

## Setup
1. Clone or download the repository
2. Navigate to the project directory
3. Ensure Python 3.13+ is installed (`python --version`)

## Running the Application
```bash
python main.py
```

## Basic Usage
1. Launch the application using the command above
2. You'll see a menu with numbered options:
   - 1. Add Task
   - 2. View Tasks
   - 3. Update Task
   - 4. Delete Task
   - 5. Mark Task Complete/Incomplete
   - 6. Exit
3. Enter the number of the option you want to perform
4. Follow the prompts to complete the operation

## Example Workflow
1. Select "1. Add Task" to create a new task
2. Enter a title and description when prompted
3. Select "2. View Tasks" to see your tasks
4. Select "5. Mark Task Complete/Incomplete" to update status
5. Select "6. Exit" when finished

## Development
- Source code is located in the `src/` directory
- Models in `src/models/`
- Services in `src/services/`
- CLI interface in `src/cli/`
- Tests in `tests/` directory
